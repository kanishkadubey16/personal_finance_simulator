# 🚀 FinSight - AI Powered Personal Finance Simulator & Advisor

## 📌 Overview
FinSight is a full-stack web application that helps users manage their finances, simulate future savings, and get AI-powered financial advice.

It allows users to track income and expenses, understand their financial habits, and make smarter decisions for the future.

---

## 💡 Features
- 💸 Track income and expenses  
- 📊 Financial simulation (predict future savings)  
- 🤖 AI-powered financial advice  
- 📈 Insights on spending habits  
- 🎯 Goal tracking (coming soon)  

---

## 🧠 How It Works
1. User enters income and expense details  
2. System calculates savings and projections  
3. AI analyzes financial data  
4. User receives personalized advice

## ⚙️ Setup Instructions

### 1. Clone the repository
```bash
git clone https://github.com/kanishkadubey16/personal_finance_simulator.git
```

### 2. Install dependencies

#### Backend
```bash
cd backend
npm install
```

#### Frontend
```bash
cd frontend
npm install
```

---

### 3. Add environment variables

Create a `.env` file inside the backend folder:

```
MONGO_URI=your_mongodb_connection
OPENAI_API_KEY=your_api_key
```

---

### 4. Run the project

#### Start Backend
```bash
npm run dev
```

#### Start Frontend
```bash
npm start
```

---
